# Project Gumpo: Feature Prioritization (MVP vs Phase 2 vs Nice-to-Have)

This document clearly separates features required for the **MVP (Pilot School Launch)** from **Phase 2 (Growth & Enhancement)** and **Nice-to-Have (Future)** features. The MVP is designed to be **completed in 8-12 weeks** and is sufficient for 1-3 schools with up to 100 students.
